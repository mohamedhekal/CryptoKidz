
# 🧠 CryptoKidz - استكشاف عالم البيتكوين

## 📖 وصف المشروع
تطبيق تعليمي تفاعلي وممتع للأطفال لتعلم أساسيات البيتكوين والتشفير بطريقة مبسطة ومرحة. يتضمن شخصية "Crypto the Cat" 🐱 التي ترشد الأطفال عبر رحلة تعليمية مثيرة.

## ✨ المميزات
- 🎮 واجهة تفاعلية ملونة ومناسبة للأطفال
- 🐱 شخصية "Crypto the Cat" المرشدة
- 🔐 شرح مبسط لإنشاء محفظة البيتكوين
- 🧩 ألعاب تعليمية تفاعلية
- 🎯 اختبارات قصيرة مع تغذية راجعة
- 🏆 نظام مكافآت وشارات
- 🔊 أصوات وتأثيرات بصرية
- 📊 تتبع التقدم والإنجازات

## 🛠️ المتطلبات
- Python 3.8 أو أحدث
- المكتبات المذكورة في `requirements.txt`

## 🚀 التثبيت والتشغيل

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. تشغيل التطبيق
```bash
python3 main.py
# أو
python3 run.py
```

### 3. اختبار التطبيق
```bash
python3 test_app.py
```

## 🚀 البدء

1. **استنساخ المستودع**
   ```bash
   git clone <رابط-المستودع>
   cd cryptokidz
   ```

2. **تثبيت المتطلبات**
   ```bash
   pip install -r requirements.txt
   ```

3. **تشغيل التطبيق**
   ```bash
   python3 main.py
   ```

4. **بدء التعلم!**
   - اختر "إنشاء محفظة" لتعلم محافظ البيتكوين
   - اختر "تعلم التشفير" لأساسيات التشفير
   - خذ الاختبارات لاختبار معرفتك
   - تحقق من تقدمك وإنجازاتك

- التمييز: #45B7D1 (أزرق فاتح)
- النجاح: #96CEB4 (أخضر فاتح)

- وسع ميزات المحفظة في `wallet_generator.py`

## 🔧 التفاصيل التقنية

### المكتبات المستخدمة
- **tkinter**: إطار عمل واجهة المستخدم
- **pygame**: الرسوم المتحركة والأصوات
- **cryptography**: العمليات التشفيرية
- **hashlib**: دوال التشفير
- **json**: تخزين البيانات
- **datetime**: تتبع الوقت

### البنية المعمارية
- **نمط MVC**: فصل النموذج-العرض-التحكم
- **التصميم المعياري**: كل ميزة في وحدات منفصلة
- **استمرارية التقدم**: تخزين البيانات بـ JSON
- **واجهة متجاوبة**: تصميم واجهة متكيف
